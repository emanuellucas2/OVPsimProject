INTRODUCTION -------------------------------------------------------
This directory contains an example from the OP Platform Modeling
Guide.

The example illustrates the use of a VLNV component library

FILES --------------------------------------------------------------
1. Two applications;
     application0/application0.c application1/application1.c
2. A VLNV library containing two modules; 
     source/ovpworld.org/module/simpleHierarchy_sub sub-system
     source/ovpworld.org/module/simpleHierarchy_top top level

BUILDING and EXECUTING ---------------------------------------------

> ./example.sh (or example.bat)
