INTRODUCTION -------------------------------------------------------
This directory contains an example from the OP Platform Modeling
Guide.

The example illustrates the use of iGen to construct an example 
module and then to use the iterator and query functions to examine
what is contained within the module.

FILES --------------------------------------------------------------
1. A Module; 
    module/module.op.tcl the module description for input to iGen
2. The harness, including the walker

BUILDING and EXECUTING ---------------------------------------------

> ./example.sh (or example.bat)
