INTRODUCTION -------------------------------------------------------
This directory contains an example from the OP Platform Modeling Guide.

The example illustrates the use of iGen to construct a platform.

FILES --------------------------------------------------------------
1. An application; application/asmtest.S
2. A module; 
    module/module.op.tcl       the hardware description for input to iGen
3. A top level testbench; 
    harness/harness.c           the control of the platform

BUILDING and EXECUTING ---------------------------------------------

> ./example.sh (or example.bat)
