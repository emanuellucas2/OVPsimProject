INTRODUCTION -------------------------------------------------------

The example shows what information can be extracted from a processor model.

FILES --------------------------------------------------------------
There are three parts to the example
1. An harness in harness/harness.c
2. Makefile to build 1.

To build the harness ----------------------------------------------

> make -C harness

RUNNING THE EXAMPLE ------------------------------------------------

> harness/harness.<IMPERAS_ARCH>.exe
