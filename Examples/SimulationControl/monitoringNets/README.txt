INTRODUCTION -------------------------------------------------------
This directory contains an example from the OP Platform Modeling
Guide.

The example illustrates the use of iGen to construct a platform.

FILES --------------------------------------------------------------
1. Two applications;
     application0/application0.c application1/application1.c
2. Two modules; 
    topmodule/module.op.tcl       the top module description for input to iGen
   submodule/module.op.tcl       the sub module description for input to iGen
3. The test harness
    harness/harness.c

BUILDING and EXECUTING ---------------------------------------------

> ./example.sh (or example.bat)
