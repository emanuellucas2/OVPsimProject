INTRODUCTION -------------------------------------------------------

This directory contains an example of an OP C++ test harness used to instance
a module.

FILES --------------------------------------------------------------
1. An application;
    application/application.c
2. A Module; 
    module/module.op.tcl  : the module description for input to iGen
3. A harness;
    harness/harness.cpp   : the C++ code to instance a module and control the
                            simulation


BUILDING and EXECUTING ---------------------------------------------

> ./example.sh (or example.bat)
